Teesside Chat Applicaiton

https://github.com/Amir-Raza-Teesside/TeessideChatApp

Teesside Chat is a social application that allows users to communicate with one another. They can share their views and opinions in the same way that they would in any other social app. To use the application, users must first sign up. The user's phone number, profile picture, and name are all that is required for validation. An OTP code is sent to the phone number for verification, allowing the user to confirm its authenticity. After registering, he user can send messages to any of the other users who are register. They can send photos and emojis. The user can also set the Status, which is visible to all members. I designed the Tees Tweet system, which allows users to write and upload Tees Tweets. All Tees Tweets will be dynamically added to the tweet Activity. All the available user of the application can read it on the Tweet Activity
